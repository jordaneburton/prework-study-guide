HTML

Defines the structure of a website

  • The head element contains info about the webpage such as metadata
  • The body element represents the visible content shown to the user
  • 'ul', 'ol', 'li' tags are for unordered and ordered lists as well as list elements

CSS

Stylizes the webpage

  • External stylesheet is the most commonly used way to style
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

Version control system, keeps track of changes to your software

  • git status: checks what branch you are on
  • git checkout -b branch-name: creates a new branch and switches to it
